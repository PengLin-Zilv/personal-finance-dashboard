# Personal Finance Dashboard

A full stack web application for tracking and analyzing personnal credit card transactions from multiple financial institutions.

This projects solves the problem of managing transactions from different banks with inconsistent CVS formats.

It automatically parses, normalizes, and categorizes spending data with visual analytics

Future Improvements:
- More institutions CVS parsing logic
- User authentications
- Machine learning for imporved categorization
- Budget tracking and alerts
- Export reports to PDF
- Mobile responsive design
- PostgreSQL for production deployment

This project was built as a portfolio piece to learn and demonstrate:
- Full-stack development skills
- Data processing and normoalization
- RESTful API design
- Modern React patterns
- Clean code architecture!!!
